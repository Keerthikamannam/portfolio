# My Portfolio
**Live Link:**
https://keerthikamannam.github.io/portfolio/
## About Portfolio
This is my personal portfolio website built as part of a university assignment.
The portfolio is designed as a **single-page website** that contains different sections such as **Home, Skills, Projects, and Contact**.
The website is mainly developed using **HTML and CSS**.
It includes navigation links, hover effects, animations, and a responsive layout to ensure the website works properly on different screen sizes.
## Home
The home page contains a section that displays my name and introduction.
Special attention was given to the design so that the large text remains responsive and properly aligned on different screen sizes.
## Skills
This section displays the technical skills I have learned.
Each skill is represented using icons and simple hover effects.
Skills included in this portfolio:
* HTML
* CSS
* Python
* Java
* Data Structures
## Projects
This section showcases the projects I have completed so far.
These projects demonstrate my interest in technology and my hands-on experience in developing applications.
## Contact
The contact section contains a simple form layout created using **HTML and CSS Flexbox**.
It allows visitors to provide their details and get in touch with me.
## Footer
The footer section contains basic information and closes the webpage with a clean design.
